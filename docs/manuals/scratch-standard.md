# 스크래치 불량 조치 기준서

## 적용 범위

제품 표면에 선형 흠집, 긁힘, 이송 방향과 유사한 마찰 흔적이 발견된 경우 적용한다.

## 주요 원인

- 지그 접촉면 마모 또는 버 발생
- 이송 레일 오염, 금속 칩, 분진 부착
- 작업대 위 이물질과 제품 표면 접촉
- 포장 전 대기 중 제품 간 마찰

## 점검 순서

1. 지그 접촉면의 마모, 날카로운 모서리, 버 발생 여부를 확인한다.
2. 이송 레일과 가이드의 오염물을 제거하고 표면 손상을 확인한다.
3. 작업대와 임시 보관 트레이의 이물질을 청소한다.
4. 동일 LOT 제품 5개 이상을 재검사해 재발 여부를 확인한다.

## 조치 기준

- 지그에 버가 있으면 즉시 사용 중지 후 교체한다.
- 레일 오염이 확인되면 청소 후 시험 운전을 수행한다.
- 동일 LOT에서 스크래치가 반복되면 해당 LOT를 격리하고 품질관리자에게 보고한다.

## 기록 항목

- 발생 설비와 LOT 번호
- 지그 교체 여부
- 레일 청소 여부
- 재검사 결과
