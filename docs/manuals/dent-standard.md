# 찍힘 불량 조치 기준서

## 적용 범위

제품 표면 또는 모서리에 눌림, 패임, 충격 흔적, 국부 변형이 확인된 경우 적용한다.

## 주요 원인

- 적재 높이 기준 초과
- 이송 속도 과다 또는 급정지
- 수동 취급 구간에서 제품 낙하
- 완충재 누락 또는 트레이 파손

## 점검 순서

1. 적재 높이가 표준 기준을 초과했는지 확인한다.
2. 이송 속도 로그와 급정지 이력을 확인한다.
3. 수동 취급 구간의 작업 방법과 낙하 가능성을 확인한다.
4. 트레이, 완충재, 보관 랙의 손상 여부를 확인한다.

## 조치 기준

- 적재 기준을 초과한 경우 즉시 재적재하고 작업자에게 기준을 재공지한다.
- 이송 급정지가 반복되면 설비 알람과 센서 상태를 점검한다.
- 완충재가 누락된 구간은 즉시 보완하고 해당 구간 제품을 추가 검사한다.

## 기록 항목

- 찍힘 위치
- 적재 상태
- 이송 로그 이상 여부
- 추가 검사 범위
