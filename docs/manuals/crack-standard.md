# 균열 불량 조치 기준서

- 문서번호: QA-STD-CRA-001
- 개정: Rev. 2026-05-23
- 적용 불량 유형: crack
- 적용 공정: 가압, 성형, 냉각, 절단, 검사 구간
- 담당: 품질관리자, 공정관리자, 설비 담당자, 현장 작업자

## 1. 목적

제품에 갈라짐, 미세 균열, 절단면 주변 파손, 하중 방향 균열이 확인될 때 생산 중지 판단, LOT 격리, 설비 조건 복원, 재검사 기준을 표준화한다. 이 기준서는 조치 Agent가 crack 불량에 대해 우선 검색해야 하는 기준 문서이다.

## 2. 적용 범위

다음 중 하나 이상에 해당하면 균열 불량으로 분류한다.

- 제품 표면 또는 단면에 선형 갈라짐이 확인된다.
- 하중 방향 또는 성형 방향을 따라 미세 균열이 확인된다.
- 절단면, 모서리, 체결부 주변에 파손 흔적이 있다.
- 동일 원자재 LOT 또는 동일 설비 조건에서 균열이 반복된다.

다음은 crack이 아니라 별도 기준서를 우선 적용한다.

- 표면 긁힘만 있고 갈라짐이 없으면 scratch 기준서를 적용한다.
- 눌림 또는 패임이 중심이면 dent 기준서를 적용한다.
- 표면 오염 또는 얼룩은 contamination 기준서를 적용한다.

## 3. 불량 등급 기준

| 등급 | 판정 기준 | 처리 기준 |
| --- | --- | --- |
| High | 기능부 균열, 관통 균열, 동일 LOT 2건 이상 반복 | 설비 정지, LOT 전량 격리, 설비 담당자 즉시 호출 |
| Medium | 외관부 미세 균열, 원자재 변경 직후 발생 | 조건 복원, 원자재 LOT 확인, 샘플 확대 재검사 |
| Low | 절단면 주변 단발성 미세 파손 | 절단 조건 확인 후 예방 점검 기록 |

## 4. 주요 원인 매트릭스

| 원인 후보 | 확인 위치 | 확인 방법 | 우선순위 |
| --- | --- | --- | --- |
| 가압 조건 이탈 | 가압 설비, 프레스, 성형기 | 압력 설정값, 알람, 최근 조건 변경 이력 확인 | High |
| 냉각 시간 부족 또는 편차 | 냉각 구간, 배출 구간 | 냉각 시간 로그, 배출 온도, 냉각수 상태 확인 | High |
| 원자재 LOT 변경 | 원자재 투입 기록 | 변경 시점과 불량 발생 시점 비교, 성적서 확인 | High |
| 금형 또는 치공구 정렬 불량 | 금형, 지그, 클램프 | 체결 상태, 편심, 마모, 정렬 기준 확인 | Medium |
| 절단 조건 불안정 | 절단날, 절단 속도 | 절단날 마모, 진동, 절단면 버 확인 | Medium |

## 5. 즉시 조치 절차

1. 균열 위치, 방향, 길이, 관통 여부를 기록한다.
2. 기능부 또는 관통 균열이면 해당 설비를 즉시 정지하고 LOT를 격리한다.
3. 가압 조건 이탈 알람과 최근 설비 설정값 변경 이력을 확인한다.
4. 냉각 시간 로그와 배출 온도를 확인한다.
5. 원자재 LOT 변경 시점과 불량 발생 시점을 비교한다.
6. 금형 정렬 상태와 치공구 체결 상태를 확인한다.
7. 조건을 표준값으로 복원한 뒤 시험 생산품을 검사한다.

## 6. 재검사 및 생산 재개 기준

생산 재개는 다음 조건을 모두 만족할 때만 가능하다.

- 가압 조건, 냉각 시간, 금형 정렬 점검 결과가 기록되어 있다.
- 조건 이탈이 확인된 경우 표준 조건으로 복원되었다.
- 시험 생산품 5개 이상에서 균열이 재발하지 않는다.
- 동일 LOT 또는 동일 원자재 LOT의 격리 상태와 판정 결과가 기록되어 있다.
- 품질관리자와 공정관리자가 재개를 승인했다.

다음 중 하나라도 발생하면 생산을 재개하지 않는다.

- 동일 원자재 LOT에서 균열이 반복된다.
- 가압 조건 또는 냉각 시간 로그가 누락되어 원인 확인이 불가능하다.
- 금형 정렬 이상이 해소되지 않았다.

## 7. 작업자 전달 문구

균열 불량이 확인되었습니다. 기능부 또는 관통 균열이면 설비를 즉시 정지하고 LOT를 격리해 주세요. 가압 조건, 냉각 시간, 원자재 LOT 변경 여부를 확인한 뒤 품질관리자와 공정관리자 승인 전까지 생산을 재개하지 마세요.

## 8. 기록 항목

- 검사 ID, LOT 번호, 공정, 설비
- 균열 위치, 방향, 길이, 관통 여부
- 가압 조건 설정값과 알람 이력
- 냉각 시간과 배출 온도
- 원자재 LOT 정보
- 금형 또는 치공구 정렬 점검 결과
- 시험 생산품 검사 결과
- 생산 재개 승인자와 승인 시각
