# 균열 불량 조치 기준서

## 적용 범위

제품에 갈라짐, 미세 균열, 절단면 주변 파손, 하중 방향 균열이 확인된 경우 적용한다.

## 주요 원인

- 가압 조건 이탈
- 냉각 시간 부족 또는 편차 증가
- 원자재 LOT 변경
- 금형 또는 치공구 정렬 불량

## 점검 순서

1. 가압 조건 이탈 알람과 최근 설비 설정값을 확인한다.
2. 냉각 시간 편차와 제품 배출 온도를 확인한다.
3. 원자재 LOT 변경 시점과 불량 발생 시점을 비교한다.
4. 금형 정렬 상태와 치공구 체결 상태를 확인한다.

## 조치 기준

- 가압 조건이 기준을 벗어난 경우 설비 조건을 복원하고 시험 생산품을 검사한다.
- 냉각 시간이 부족하면 표준 시간으로 조정하고 재발 여부를 확인한다.
- 동일 원자재 LOT에서 균열이 반복되면 원자재 LOT를 격리한다.

## 기록 항목

- 균열 위치와 방향
- 가압 조건 기록
- 냉각 시간 기록
- 원자재 LOT 정보
